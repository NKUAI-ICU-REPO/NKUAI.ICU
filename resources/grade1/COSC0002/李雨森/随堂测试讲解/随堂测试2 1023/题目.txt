题目描述
12个人围坐成一圈（编号顺时针方向依次为1~12），从1号开始数，沿顺时针方向，当数到k时，则该人被淘汰，接着从被淘汰的人下一位继续从1开始数起，数到k时淘汰第二人，依此类推，知道剩下最后一人。输出最后一人的编号，以及每一轮的结果。

输入
输入淘汰计数k，k>1。

输出
输出最后一人的编号，以及每一轮的结果。注意编号范围为1~12。

样例输入
8
样例输出
  1  2  3  4  5  6  7  0  9 10 11 12
  1  2  3  0  5  6  7  0  9 10 11 12
  0  2  3  0  5  6  7  0  9 10 11 12
  0  2  3  0  5  6  7  0  9 10  0 12
  0  2  3  0  5  6  7  0  9  0  0 12
  0  2  3  0  5  6  7  0  9  0  0  0
  0  2  0  0  5  6  7  0  9  0  0  0
  0  2  0  0  5  6  0  0  9  0  0  0
  0  2  0  0  5  0  0  0  9  0  0  0
  0  0  0  0  5  0  0  0  9  0  0  0
  0  0  0  0  5  0  0  0  0  0  0  0
  5
提示
k的取值范围为int可表示的正整数，无需判断输入是否合法。
输出时，使用setw(3)将每个数字的输出宽度限制为3，以便查看。
